README

On associe au rapport du projet Open Science un dossier .zip contenant un script R, 4 fichiers .csv contenant les données nécessaires pour les analyses et 2 fichiers d’arbres phylogénétiques pour les chauves-souris et pour les plantes. 
Il faut faire tourner le code en vérifiant que tous les packages sont bien installés.
Le code renvoie 4 fichiers .pdf contenant les représentations graphiques du signal phylogénétique en fonction de différents paramètres, et les régressions linéaires. On notera que la lettre A désigne les plantes et B les chauves-souris dans toutes les analyses. 
Le code renvoie également un fichier csv qui contient le résultats de tous les tests de Mantel des 43 communautés locales.  
Enfin l’étape 6 du code contient tous les tests statistiques que nous avons réalisés pour plus de détails méthodologiques.
